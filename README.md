# Desktop Buddy

A transparent desktop widget that displays a looping webm video - like a desktop pet!

## Features

- **Transparent window** - only the video is visible, no background
- **Always on top** - stays above other windows
- **Draggable** - click and drag anywhere on the video to move it
- **Resizable** - scroll wheel or drag the corner to resize
- **Supports** webm, mp4, gif, and webp files

## Getting started

```bash
npm install
npm start
```

## Usage

1. Run the app and a dropzone will appear
2. Drag & drop your webm file onto it, or click to browse
3. Drag the video anywhere on your screen
4. Scroll wheel to resize
5. Hover to see controls (close, load new video, reset size)
